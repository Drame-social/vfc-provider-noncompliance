# Resume Bullets

## Public Health Data Analyst

Built a GitHub-ready synthetic VFC-style provider non-compliance analytics project using Python, SQL, SAS, and Power BI methods to generate validated site-visit datasets, perform data-quality checks, and produce annual/domain-level compliance reporting outputs.

## Epidemiologist

Developed a synthetic 12-budget-period provider compliance monitoring analysis to examine non-compliance trends by region, provider category, urbanicity, visit method, provider size, and time in program, using complete-case cohort logic, chi-square testing, and adjusted logistic regression framework.

## Public Health Informatics / Data Modernization

Designed a reproducible public health data pipeline with synthetic provider/site-visit tables, data dictionary, validation rules, SQL schema, SAS analysis, and dashboard-ready outputs to demonstrate secure data modernization practices without exposing confidential CDC or provider data.
